===== 1. WHAT IS MONGODB =====
It is a no SQL database management system that can manage a humongous amount of data.
It stores data as BSON (Binary JavaScript Object Notation).


COLLECTIONS -> TABLES
DOCUMENTS ->  ROWS OF DATA



Collections are synonymous to sql tables.
Documents are synonymous to sql rows.
Data is stored in JSON-like format (BSON)

Every document must have a "_id" field similar to sql primary key, either provided or auto generated by mongodb. It identifies the document uniquely.

Document maximum collection: 16MB.
